## Setup environment

```
// setting config file pada direktori `/config/base.php` untuk koneksi database

define("HOST", "<hostname>");
define("DBNAME", "<database_name>");
define("USERNAME", "<username>");
define("PASSWORD", "<password>");


// setting config file pada direktori `/config/url.php` untuk path dan url website

define("BASEURL", "<url_website>");
define("BASEASSET", "<url_website>/assets");
define("BASEPATH", $_SERVER['DOCUMENT_ROOT'] . "/ta-kelompok-paw");
```
